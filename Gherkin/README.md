## Exercícios de Gherkin

Esta pasta reúne os exercícios desenvolvidos durante o módulo de **BDD (Behavior Driven Development)** do curso de **Engenharia de Qualidade de Software da EBAC**.

Os arquivos `.feature` foram elaborados a partir de histórias de usuário e critérios de aceitação, utilizando a sintaxe Gherkin para descrever o comportamento esperado da aplicação de forma clara, legível e reutilizável.

Os cenários contemplam:

- Fluxos principais e alternativos;
- Critérios de aceitação;
- Contexto compartilhado (`Contexto`);
- Cenários (`Cenário`);
- Esquema do Cenário (`Esquema do Cenário`) com tabelas de exemplos (`Exemplos`).

### Arquivos

- `login.feature`
- `checkout.feature`
- `config.feature`
