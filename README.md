# 🧪 Quality Assurance(QA) 

Repositório dedicado aos projetos desenvolvidos durante o curso **Engenharia de Qualidade de Software (QA)** da **EBAC (Escola Britânica de Artes Criativas e Tecnologia)**.

Este repositório reúne atividades práticas, desafios e projetos aplicando conceitos fundamentais de Quality Assurance, com foco na qualidade de software e na evolução das boas práticas de testes.

Ao longo do curso, são abordados temas como:

- Testes manuais
- BDD (Behavior Driven Development)
- Gherkin
- Histórias de usuário e critérios de aceitação
- Casos de teste
- Planejamento e execução de testes
- Relato e acompanhamento de defeitos (Bug Report)
- Testes de API
- Testes de interface (UI)
- Automação de testes
- Versionamento com Git e GitHub
- Boas práticas de Quality Assurance

O objetivo deste repositório é documentar minha evolução durante a formação e servir como um portfólio dos projetos desenvolvidos ao longo do curso.
